# Smart Finance App

Personal finance mobile app built with Flutter and Node.js.

## 🚀 Tech Stack

- Flutter (Mobile)
- Node.js + Express (Backend)
- PostgreSQL (Database)

## 📁 Structure

- /backend → API
- /mobile → Flutter app
- /docs → documentation

## 🎯 Features

- Authentication (JWT)
- Expense tracking
- Clean architecture

## 🛠️ Setup

### Backend
cd backend
npm install
npm run dev

### Mobile
cd mobile/app
flutter pub get
flutter run
