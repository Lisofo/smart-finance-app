# 💸 Smart Finance App

A fullstack personal finance application built with Flutter and Node.js.

Track your expenses, understand your spending habits, and take control of your finances.

---

## 🚀 Tech Stack

### 📱 Mobile

* Flutter
* Clean Architecture
* Provider (State Management)
* GoRouter (Navigation)

### 🌐 Backend

* Node.js
* Express
* PostgreSQL
* JWT Authentication
* REST API

---

## 📁 Project Structure

```
smart-finance-app/
├── backend/   # Node.js API
├── mobile/    # Flutter app
├── docs/      # Documentation & assets
```

---

## ✨ Features

* 🔐 User Authentication (Register / Login)
* 💰 Expense Tracking
* ✏️ Create, Edit, Delete expenses
* 📊 Clean architecture (scalable)
* 🔒 Secure storage for tokens

---

## 🛠️ Getting Started

### Backend

```bash
cd backend
npm install
npm run dev
```

---

### Mobile

```bash
cd mobile/app
flutter pub get
flutter run
```
